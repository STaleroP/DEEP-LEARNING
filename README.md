# Physical Modeling

A collection of Jupyter notebooks and code for computational physics and
scientific machine learning, developed as coursework and lab assignments.

## Contents

| Folder | Topics |
|---|---|
| `0.Introduccion` | Course introduction and definitions |
| `1.EDOs` | ODE methods: Euler, RK4, RK45, Adams–Bashforth–Moulton, stability analysis |
| `2.Numerov-ESIT` | Numerov method for the TISE; quantum potential wells and eigenvalue problems |
| `3.EDPs` | PDE solvers: finite differences (Python, C, Fortran); wave and Poisson equations |
| `4.Redes_Neuronales` | Deep learning with PyTorch/Keras: backpropagation, CNNs, RNNs, LSTMs, transfer learning |
| `5.PINNs` | Physics-Informed Neural Networks for heat, Laplace, and Poisson equations |
| `6.QC` | Quantum computing: linear algebra review, quantum gates, Deutsch and Deutsch–Jozsa algorithms |
| `NN-Homeworks` | Applied neural network projects: image classification, parachute dynamics with LSTM |

## Stack

- **Python 3.13** · NumPy · Matplotlib · SciPy
- **PyTorch 2.x** · Keras 3 (PyTorch backend)
- **Qiskit** (quantum computing notebooks)
- C and Fortran for low-level PDE solvers
